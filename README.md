# rzd

## Технологии

- **Фронтенд**: React
- **Бекенд**: Spring Boot
- **База данных**: PostgreSQL

## Что умеет Api

### Клиент
- Может получать маршуты, в заданный день отправления
- Получать информацию о местах (цена, доступность), в выбранном маршруте поезда
- Выбирать и бронировать места
